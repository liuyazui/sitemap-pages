Cursor入门指南

欢迎使用Cursor入门指南！本文将帮助你了解如何安装、配置和开始使用这款智能代码编辑器。 Cursor结合了VSCode的强大功能和现代AI技术，为你提供前所未有的编程体验。 让我们一起开始这段AI辅助编程的旅程吧！

系统要求

在安装Cursor之前，请确保你的系统满足以下要求：

Windows

  • Windows 10/11 64位
  • 4GB RAM (建议8GB以上)
  • 2GHz双核处理器
  • 1GB可用磁盘空间
  • DirectX 9或更高版本

macOS

  • macOS 10.15 Catalina或更新版本
  • 4GB RAM (建议8GB以上)
  • Intel或Apple Silicon处理器
  • 1GB可用磁盘空间

Linux

  • Ubuntu 18.04+, Debian 10+, CentOS 7+, Fedora 32+
  • 4GB RAM (建议8GB以上)
  • 2GHz双核处理器
  • 1GB可用磁盘空间
  • GNOME, KDE或其他现代桌面环境

安装Cursor

Windows安装

  1. 访问Cursor官网下载Windows安装包
  2. 运行下载的.exe文件
  3. 按照安装向导的指示完成安装
  4. 安装完成后，Cursor会自动启动

macOS安装

  1. 访问Cursor官网下载macOS安装包
  2. 打开下载的.dmg文件
  3. 将Cursor应用拖到Applications文件夹
  4. 从Applications文件夹或Launchpad启动Cursor
  5. 首次启动时可能需要授权运行

Linux安装

  1. 访问Cursor官网下载适合你发行版的安装包
  2. Debian/Ubuntu: sudo dpkg -i cursor_X.X.X_amd64.deb
  3. Fedora/CentOS: sudo rpm -i cursor-X.X.X.x86_64.rpm
  4. 或解压.tar.gz文件并运行可执行文件

首次启动

首次启动Cursor时，你会看到欢迎界面，引导你完成初始设置：

  1. 登录或创建账号 - 使用GitHub账号登录或创建新的Cursor账号
  2. 选择主题 - 选择亮色或暗色界面主题
  3. 选择键盘快捷键方案 - 默认、VSCode、Vim等
  4. 选择常用编程语言 - 根据你的需求选择
  5. AI配置 - 设置使用的AI模型和参数

完成初始设置后，你将看到Cursor的主界面。你可以打开现有项目或创建新项目开始使用。

用户界面概览

Cursor的界面布局与VSCode类似，但增加了AI功能区域：

主要界面元素

  • 活动栏 - 左侧垂直图标栏，用于切换主要视图
  • 侧边栏 - 可显示文件浏览器、搜索、Git控制等
  • 编辑器区域 - 中央代码编辑区域，可分为多个分栏
  • 状态栏 - 底部显示当前状态、行号、编码等信息
  • AI面板 - 可通过Ctrl+J(Windows/Linux)或Cmd+J(Mac)打开的AI交互面板

AI专属界面元素

  • AI聊天面板 - 与AI助手对话的界面
  • 代码生成区域 - 显示AI生成的代码建议
  • 智能补全 - 在编辑器中直接显示的AI代码补全

基本使用

打开项目

你可以通过以下方式打开项目：

  • 点击"File > Open Folder"菜单选项
  • 使用快捷键Ctrl+O (Windows/Linux)或Cmd+O (Mac)
  • 从最近项目列表中选择

创建文件

创建新文件的方法：

  • 在文件浏览器中右键点击，选择"New File"
  • 使用快捷键Ctrl+N (Windows/Linux)或Cmd+N (Mac)
  • 点击"File > New File"菜单选项

保存文件

保存文件的方法：

  • 使用快捷键Ctrl+S (Windows/Linux)或Cmd+S (Mac)
  • 点击"File > Save"菜单选项
  • 自动保存功能会定期保存你的更改

AI功能入门

与AI对话

要开始与AI助手对话：

  1. 按下Ctrl+J (Windows/Linux)或Cmd+J (Mac)打开AI面板
  2. 在输入框中输入你的问题或请求
  3. 按Enter发送消息
  4. AI会根据你的项目上下文提供回答

基本代码生成

让AI生成代码的方法：

  1. 在AI面板中描述你需要的功能，例如"创建一个计算斐波那契数列的函数"
  2. AI会生成相应代码并展示在对话中
  3. 点击代码块右上角的"插入到编辑器"按钮将代码添加到当前文件

解释代码

获取代码解释的方法：

  1. 在编辑器中选中要解释的代码
  2. 右键点击并选择"Explain Selection"
  3. 或在AI面板中输入"解释这段代码："后粘贴代码
  4. AI会提供详细解释，包括功能、逻辑和潜在问题
